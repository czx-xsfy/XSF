XSF_MessageBox

int XSF_MessatBox(std::optional<XSF_Window> window, std::wstring text, std::wstring caption, int message_type, int language)

MessageBoxEx 的轻量封装。实现中的函数名为 XSF_MessatBox；文档中使用 MessageBox 为更直观的命名。可传入 optional 窗口作为父窗口句柄；返回 MessageBoxEx 的返回码。

示例

int rc = XSF_MessatBox(std::nullopt, L"确认退出？", L"提示", MB_OKCANCEL, XSF_LANG_SYSTEM);
